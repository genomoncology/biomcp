# Use Case: Drug Shortage Monitoring

Monitor FDA drug shortages for supply chain planning and clinical decisions.

## Workflow

1. **Search current shortages** - List drugs currently in shortage
2. **Search by drug name** - Check specific drug status
3. **Filter by category** - Focus on therapeutic area
4. **Get shortage details** - Reason, resolution, alternatives

## Shortage Status Types

| Status             | Description                     |
| ------------------ | ------------------------------- |
| Current            | Active shortage                 |
| Resolved           | Shortage ended, supply restored |
| To Be Discontinued | Product being withdrawn         |

## Common Shortage Reasons

- Demand increase
- Manufacturing issues
- Raw material shortage
- Regulatory action
- Business decision

## Tips

- Check shortages before prescribing high-risk medications
- Monitor oncology drugs closely (frequent shortages)
- Contact manufacturers directly for supply timelines
